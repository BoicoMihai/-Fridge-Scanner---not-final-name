## What is Fridge Scanner?

**Fridge Scanner** turns your leftover ingredients into delicious, ready-to-cook recipes — no meal planning stress, no wasted food, no "I don't know what to make" moments.

Type in what you have. Get a recipe. Start cooking.

---

## Features

| Feature | Description |
|---|---|
| **Smart Search** | Search recipes by ingredient with autocomplete suggestions |
| **Random Discovery** | Get a surprise recipe with one click |
| **Dietary Filters** | Vegan · Vegetarian · Gluten-Free · Dairy-Free · Paleo · Keto |
| **Cooking Time Slider** | Filter recipes by how long you actually have |
| **Favorites** | Save recipes you love and revisit them anytime |
| **Nutrition Breakdown** | Calories, protein, carbohydrates, and fat — at a glance |
| **Search History** | Your past searches, always one click away |
| **Fully Responsive** | Works on mobile, tablet, and desktop |

## Getting Started

No installation. No setup. No dependencies.

**Option 1 — Open directly in your browser:**
```
Open index.html in any modern browser
```

**Option 2 — Use Live Server (VS Code):**
```
Port 5501 is already configured in .vscode/settings.json
```

**Option 3 — Visit the live site:**
```
Check the Deployments section of this repository for the latest link or join here
https://fridge-scanner-app.vercel.app/
```

## Built With

- Vanilla HTML, CSS, and JavaScript — no frameworks, no bundlers
- [Spoonacular API](https://spoonacular.com/food-api) — recipe search, autocomplete, and nutrition data
- Google Fonts — Playfair Display · DM Sans · Afacad · Alkatra
- localStorage — favorites and history persist between sessions
